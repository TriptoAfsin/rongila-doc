### Using rows and columns in Rongila CSS is a breeze

### Example - 

```html
    <div class="row auto-spacing">
        <div class="column">
            <h1>Heading 1</h1>
            <h2>Heading 2</h2>
            <h3>Heading 3</h3>
        </div>
        <div class="column mt2">
            <p class="txt-xl">Extra Large Text</p>
            <p class="txt-lg">Large Text</p>
        </div>
        <div class="column">
            <p class="title1">Title 1</p>
        </div>
    </div>
```
