** Any color can be used as a page theme in rongila css **

**But we recommend these colors as site themes - ** 
1. Dark Theme: ``` darkBg ```
2. Light Theme ``` LightBg ```
3. Grey Theme: ``` grey ```

### How to use - 

**1. Dark Theme - **
```html
    <body class="darkBg">
        <h1>Dark Theme</h1>
    <body>
```
